Sudoku Solver Program

This program solves Easy, Medium, and Moderately Hard sudoku problems through a recursive function method.

  • By converting the 9x9 grid into a matrix, I was easily able to call rows and columns efficiently. I then used two functions to firstly identify viable inputs into the puzzle and then append those inputs into the matrix. Currently working on resolving an issue regarding duplicate numbers in the nine 3x3 squares while solving hard puzzles.

Morse Code Communication in Micro:bit

This program allows Micro:bits to send and receive messages encoded in Morse Code.

  • Created a dictionary using the Morse Code alphabet as the "key" and the standard English alphabet as the "value." I then had this dictionary return the string "value" once the user entered a valid "key." This implementation allowed for quick and efficient communication.
